Arquitectura de Computadoras

La Arquitectura de Computadoras es una disciplina fundamental que estudia el diseño, organización y funcionamiento interno de los componentes esenciales de una computadora, así como su interacción para procesar información de manera eficiente.

Esta materia aborda los elementos clave de un sistema computacional:

  • CPU - Unidad Central de Procesamiento, encargada de ejecutar instrucciones
  • Memoria RAM - Almacena datos y programas en ejecución
  • Dispositivos de almacenamiento - Discos duros y SSDs que guardan información permanentemente
  • Tarjeta madre - Placa base que interconecta todos los componentes
  • Periféricos - Dispositivos de entrada y salida para la comunicación con el usuario
Jerarquía de Memoria

Un aspecto clave es la jerarquía de memoria, que va desde los registros y caché (los más rápidos pero de menor capacidad) hasta la RAM y el almacenamiento secundario (más lentos pero con mayor espacio).

Ciclo de Instrucción

Se estudia cómo la CPU gestiona las instrucciones mediante ciclos de fetch, decode, execute y store, así como técnicas de optimización como el pipeline y las diferencias entre arquitecturas CISC y RISC.

Componentes Principales

La arquitectura de computadoras se centra en comprender cómo interactúan estos componentes para ejecutar programas y procesar datos de manera eficiente. Cada elemento tiene un rol específico en el flujo de información dentro del sistema.

Optimización del Rendimiento

Los arquitectos de computadoras buscan constantemente formas de mejorar el rendimiento mediante técnicas como el paralelismo, la segmentación (pipelining) y la jerarquía de memoria, equilibrando velocidad, capacidad y costo.
